<p>
A HoldIT Game Engine
Features:
<ul>
  <li>Loading .obj models w/ materials</li>
  <li>Loading RGB and RGBa textures</li>
  <li>Real time rendering of multiple objects</li>
  <li>Rendering using a perspective first-person camera</li>
</ul>
